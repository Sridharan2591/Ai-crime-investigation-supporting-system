# AI Powered Digital Forensics & Crime Investigation Supporting System

## Overview

The AI Powered Digital Forensics & Crime Investigation Supporting System is an intelligent forensic analysis platform designed to assist in identifying digital evidence manipulation and analyzing suspicious behavior. The system integrates multiple Artificial Intelligence and Machine Learning models into a single web-based dashboard.

This project combines image tampering detection, video tampering detection, audio tampering detection, facial emotion recognition, voice emotion analysis, and an offline AI chatbot to support digital forensic investigations.

---

## Features

### Image Tampering Detection

* Detects manipulated or forged images.
* Identifies inconsistencies in textures, edges, and image patterns.

### Video Tampering Detection

* Analyzes video frames for signs of manipulation.
* Detects frame-level inconsistencies and suspicious modifications.

### Audio Tampering Detection

* Identifies edited or manipulated audio recordings.
* Uses audio feature extraction techniques for analysis.

### Facial Emotion Analysis

* Detects facial expressions and predicts emotions.
* Supports image-based and live webcam analysis.

### Voice Emotion Analysis

* Predicts emotions from speech signals.
* Detects emotions such as Happy, Angry, Fear, and Neutral.

### Offline AI Chatbot

* Integrated with Ollama.
* Provides offline AI assistance and interaction.

### Suspicion Score Generation

* Combines outputs from multiple modules.
* Generates a suspicion score based on detected anomalies.

### Report Generation

* Generates investigation reports automatically.
* Summarizes all analysis results in a structured format.

---

## Modules

1. Image Tampering Detection
2. Video Tampering Detection
3. Audio Tampering Detection
4. Facial Emotion Analysis
5. Voice Emotion Analysis
6. Offline AI Chatbot
7. Suspicion Score Generation
8. Automated Report Generation

---

## Project Workflow

1. User uploads image, video, or audio evidence.
2. Flask routes the request to the appropriate AI module.
3. The AI model analyzes the input data.
4. Results are returned to the dashboard.
5. Suspicion score is calculated.
6. Investigation report is generated.
